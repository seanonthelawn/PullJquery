F = GMm/R2

This is my Javascript as of 3/1/2017. Some things id like to point out are the tabs, the lines that assign and remove the selected class, and the math function.
The tabs...
Asigning and removing the selected class allows me to differentiate between which tabs, and assign CSS propertys to the active tab that are not present for the two inactive tabs.
The math function pulls all of the information from the form fields and assigns them to variables. Then it plugs the variables into a print function with the gravitational attraction equation in it. Notably, the equation divides by the distance squared, so if you type 0 into the distance field in the form, the equation will return infinity.
